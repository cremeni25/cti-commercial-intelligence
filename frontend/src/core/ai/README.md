# CTI Core AI

Núcleo central de inteligência artificial do CTI.

Responsabilidades:
- IA corporativa
- análises globais
- insights comerciais
- predição operacional
- scoring inteligente
- automações
- inferência estratégica
- recomendação operacional

Regras:
- não conter lógica específica de domínio
- não conter UI
- não conter páginas
- não conter componentes visuais

Toda IA específica de domínio deve nascer em:
modules/[dominio]/ai
